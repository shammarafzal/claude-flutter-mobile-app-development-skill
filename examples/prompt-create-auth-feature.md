# Example Prompt: Create Auth Feature

## Prompt

Create a feature-first Clean Architecture auth module using Cubit, Dio, and Laravel API.

Include login, register, forgot password, token storage, API error handling, and logout.

## Expected Claude output

- `features/auth/data`
- `features/auth/domain`
- `features/auth/presentation`
- User entity
- User model
- Auth repository
- Remote/local data sources
- Login/register use cases
- Auth Cubit and states
- Login page

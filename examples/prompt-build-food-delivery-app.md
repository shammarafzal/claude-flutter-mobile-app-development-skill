# Example Prompt: Build a Food Delivery App

## Prompt

Build a Flutter app architecture for a food delivery app with Laravel backend.

The app needs customer login, restaurant listing, menu, cart, checkout, order tracking, profile, push notifications, and admin-managed data.

Use feature-first Clean Architecture, Dio, Cubit, and Laravel REST API.

## Expected Claude output

- App scope summary
- Feature list
- Screen list
- API/backend requirements
- Folder structure
- Recommended packages
- Development milestones
- First files to create

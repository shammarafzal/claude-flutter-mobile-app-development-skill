# Contributing

Thanks for your interest in improving this community Claude Skill.

## How to contribute

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Keep examples practical and Flutter-focused.
5. Open a pull request with a clear explanation.

## Contribution ideas

- Add new workflow resources
- Improve Flutter architecture guidance
- Add example prompts
- Add templates for Riverpod, Bloc, Firebase, Supabase, or testing
- Improve publishing checklists
- Fix typos or unclear instructions

## Guidelines

- Do not present this project as official Flutter, Google, Anthropic, or Claude content.
- Avoid copying third-party content without proper license notices.
- Keep the Skill practical and production-focused.
